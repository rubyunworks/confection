== Verify

Lets say we have a confection configuration file `confile.rb` containing:

  foo do
    @a = 1
  end

We can get hold of the configuration for 'foo' using the `#confection`
method.

  config = confection(:foo)

  Confection::Controller.assert === config

Then if we execute it using `#exec` it will be evaluate in the current
context.

  config.exec

  @a  #=> 1
