== Verify

Lets say we have a confection configuration file `conf.rb` containing:

  foo do
    @a = 1
  end

We can get hold of the configuration for 'foo' using the `#confection`
method.

  conf = confection(:foo)

  Confection::Controller.assert === conf

Then if we execute it using `#exec` it will be evaluate in the current
context.

  conf.exec

  @a  #=> 1
